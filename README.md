# wordset
Гибкий контейнер для хранения уникальных слов с возможностью переключения между HashSet и TreeSet. 4 стратегии хеширования (по длине, гласным, разнице кодов, сумме символов) и 5 классов-компараторов для сортировки. Строго соблюдён контракт equals/hashCode. Проект демонстрирует понимание внутреннего устройства коллекций и принципов SOLID.
