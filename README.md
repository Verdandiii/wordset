# WordSet Engine — библиотека для анализа и индексации текста

[![Java Version](https://img.shields.io/badge/Java-17%2B-blue)](https://www.oracle.com/java/)
[![JUnit](https://img.shields.io/badge/JUnit-5-green)](https://junit.org/junit5/)

## О проекте

Учебный проект в рамках курса «Прикладная математика и информатика» (ЮФУ). Представляет собой гибкий контейнер для хранения уникальных слов с возможностью переключения между `HashSet` и `TreeSet` и выбора стратегии хеширования или сортировки.

### Зачем это нужно?

Представьте, что вы пишете поисковый движок или анализатор текста. Вам нужно хранить уникальные слова. Иногда важен быстрый поиск (HashSet), иногда — упорядоченный вывод (TreeSet). WordSet позволяет переключаться между ними «на лету», не меняя остальной код.

## Стратегии хеширования (HashSet)

| Стратегия | `enum HashType` | Правило |
| :--- | :--- | :--- |
| По длине слова | `LENGTH` | `word.length()` |
| По количеству гласных | `VOWELS_COUNT` | Подсчёт a, e, i, o, u |
| По разнице кодов первой и последней буквы | `FIRST_LAST_DIFF` | `charAt(0) - charAt(last)` |
| По сумме кодов всех букв | `SUM_CODES` | Сумма `char`-значений |

Важно: `hashCode` может совпадать у разных слов (коллизия), но `equals` различает их по содержимому. Это корректное поведение для HashSet.

## Компараторы (TreeSet)

| Компаратор | Класс | Порядок сортировки |
| :--- | :--- | :--- |
| Естественный | `Comparable` (compareTo) | Алфавитный |
| По длине | `ByLengthComparator` | От коротких к длинным |
| По первым трём символам | `ByFirstThreeCharsComparator` | Лексикографический по префиксу |
| По уникальным символам | `ByUniqueCharsCountComparator` | По количеству разных букв в слове |
| Обратный алфавитный | `ReverseAlphabetComparator` | От Z до A |
| По гласным | `ByVowelCountComparator` | По числу гласных букв |

## Быстрый старт

### Требования
- Java 17 или выше
- Любая IDE (IntelliJ IDEA, Eclipse) или командная строка

### Запуск из командной строки

```bash
# 1. Клонировать репозиторий
git clone https://github.com/Verdandiii/wordset-engine.git

# 2. Перейти в папку с исходниками
cd wordset-engine/src

# 3. Скомпилировать
javac Main.java

# 4. Запустить
java Main
